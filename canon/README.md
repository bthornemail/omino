# Canon Index

This directory imports canonical source documents as separate files. They are not silently concatenated.

## Resolution Rules

```text
newer explicit canonical statement overrides older exploratory wording
implementation does not automatically override canon
projection wording does not define validation authority
record disagreements instead of erasing them
```
