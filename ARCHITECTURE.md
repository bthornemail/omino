# Architecture

Omino packages the native runtime and projection surfaces for:

```text
Emergent Axial Lisp
Meta-CONS Runtime
Omnicron Coproduct Partition
LOGOS/NOMOS/PATHOS Hamming-Coded COBS-CONS
OMNION deterministic clock and centroid
```

## Runtime Semantics

- `.o` files declare.
- Emergent Axial Lisp resolves.
- Meta-CONS composes.
- Hamming/Miquel checks integrity.
- COBS frames.
- CONS preserves continuation.
- Validation accepts.
- Attestation records.
- HTML, Canvas, DOT, and SVG project.

## Preserved Laws

- Origin-preserving coproduct.
- Non-destructive overlap.
- Validated Union-Find only.
- 256-position OMI-Lisp plane.
- Nibble interleave `0xYX -> 0xY0X0`.
- Selectors `0..3` route Local/CAR.
- Selectors `4..7` route Remote/CDR.
- Remote selector base equals Local selector base XOR `0x80`.
- LOGOS/NOMOS/PATHOS checks integrity.
- OMNION centroid and deterministic clock remain fixed authorities for synchronization.

## Projection Boundary

Projection code may render native state, but must retain source provenance and validation state. Projection output must not collapse nodes merely because visible addresses match.
